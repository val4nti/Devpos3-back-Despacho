# Backend Despacho

## Descripción

Microservicio desarrollado con Spring Boot encargado de administrar los despachos del sistema mediante una API REST.

## Tecnologías

- Java
- Spring Boot
- Maven
- MySQL

## Instalación

```bash
git clone https://github.com/val4nti/Devpos3-back-Despacho.git
cd Devpos3-back-Despacho
mvn clean install
mvn spring-boot:run
```

## Funcionalidades

- Registrar despachos.
- Consultar despachos.
- Actualizar estados.
- Eliminar registros.
